## 2. 组件规范

### 2.1 顶部栏
| 属性 | 值 |
|------|-----|
| 高度 | 48px |
| 背景 | 纯白 `#FFFFFF` |
| 底部边框 | 1px `gray-200` |
| 按钮 | Lucide Icons 24px, `gray-600` |

### 2.2 标题区域
| 元素 | 样式 |
|------|------|
| 标题 | 24px, `gray-900`, 600 |
| 元信息 | 12px, `gray-500`, 400 |
| 间距 | 标题下 8px，元信息下 16px |

### 2.3 音频播放器
| 属性 | 值 |
|------|-----|
| 背景 | `gray-50` |
| 高度 | 80px |
| 进度条 | 2px, `indigo-600` |
| 控制按钮 | Lucide Icons 24px |
| 播放按钮 | 48px 圆形，`indigo-600` |

### 2.4 AI 摘要卡片
| 属性 | 值 |
|------|-----|
| 背景 | 纯白 |
| 边框 | 1px `gray-200` |
| 圆角 | 8px |
| 内边距 | 16px |
| 图标 | `sparkles` 20px, `indigo-600` |

### 2.5 关键信息网格
| 属性 | 值 |
|------|-----|
| 布局 | 2×2 网格（移动）/ 4×1（桌面） |
| 卡片高度 | 80px |
| 背景 | 纯白 |
| 边框 | 1px `gray-200` |
| 圆角 | 8px |

### 2.6 转录文本区
| 属性 | 值 |
|------|-----|
| 背景 | 透明 |
| 字体 | 16px, 1.5 行高 |
| 颜色 | `gray-800` |
| 时间戳 | 12px, `indigo-600`, 加粗 |

| 属性 | 值 |
|------|-----|

| 圆角 | 8px |

## 3. 交互流程

- **音频播放**：播放/暂停/跳转/点击时间戳跳转
- **文本编辑**：点击编辑按钮进入编辑模式
- **AI 摘要**：录音完成自动处理，可展开/编辑
- **待办提取**：AI 识别待办，点击复选框标记完成
