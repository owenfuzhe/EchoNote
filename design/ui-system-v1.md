## 1. 设计原则

### 1.1 产品定位

**EchoNote** 是一款语音优先的智能笔记应用，核心价值：
- 🎙️ **语音输入**：3 秒启动录音，端侧原生识别
- 🤖 **AI 智能**：Kimi/GLM 驱动，自动摘要 + 标签
- ☁️ **云端同步**：默认云端存储，多设备无缝切换
- 💰 **免费 + 付费**：基础功能免费，AI 高级功能付费

### 1.2 设计关键词

| 关键词 | 含义 | 设计体现 |
|--------|------|---------|
| **极简** | 减少认知负担 | 留白充足，信息分层清晰 |
| **快速** | 3 秒启动录音 | 核心操作一键可达 |
| **智能** | AI 辅助但不打扰 | 智能功能可感知但克制 |
| **信任** | 数据隐私透明 | 清晰的隐私指示 + 本地处理标识 |

### 1.3 品牌个性

```
专业但不冰冷 ○────────● 活泼但不轻浮
              ↓
        专业工具感 + 适度亲和
```

**推荐方向：** 偏向专业工具感（类似 Notion），但加入适度的温暖元素（圆角、柔和色彩）

## 2. 风格方向

提供 3 个风格方向供选择：

### 选项 A：专业工具感（推荐 ⭐）

**参考：** Notion、Obsidian、Bear

**特点：**
- 中性色为主（灰/白/黑）
- 强调色克制使用
- 信息密度适中
- 圆角适中（8-12px）
- 阴影细腻

**适合人群：** 知识工作者、学生、专业人士

**视觉示意：**
```
┌─────────────────────────────────┐
│  EchoNote                       │ ← 简洁顶部栏
├─────────────────────────────────┤
│  ╭─────────────────────────╮   │
│  │   🎙️ 点击录音          │   │ ← 大按钮，中性色
│  │   3:24                  │   │
│  ╰─────────────────────────╯   │
│                                 │
│  最近笔记                       │
│  ┌─────────────────────────┐   │
│  │ 会议记录                │   │ ← 卡片式，细边框
│  │ 2 分钟前 • AI 摘要...    │   │
│  └─────────────────────────┘   │
└─────────────────────────────────┘
```

**配色预览：**
- 主色：Indigo (#4F46E5) — 专业、信任
- 背景：White / Gray-50
- 文字：Gray-900 / Gray-600

---

### 选项 B：活泼亲和

**参考：** Flomo、Day One、Notion(彩色版)

**特点：**
- 明亮色彩点缀
- 圆角更大（12-16px）
- 表情符号丰富
- 动画更生动
- 插画风格轻松

**适合人群：** 年轻用户、创意工作者

**视觉示意：**

**参考：** Apple Notes、Things 3、Cron

**特点：**

- 单色或双色系统
- 动画极其克制

**适合人群：** 极简主义者、苹果生态用户

### 风格对比总结

| 维度 | 选项 A 专业工具 | 选项 B 活泼亲和 | 选项 C 极简克制 |
|------|---------------|---------------|---------------|
| 主色 | Indigo | Purple | Black |
| 圆角 | 8-12px | 12-16px | 4-8px |
| 阴影 | 细腻 | 明显 | 几乎无 |
| Emoji | 克制使用 | 丰富 | 不使用 |
| 动画 | 适中 | 生动 | 极简 |
| 推荐度 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐ |

**Designer 推荐：选项 A（专业工具感）**

理由：
1. 符合"工具"定位，用户心智明确
2. 色彩克制，长期使用不疲劳
3. 扩展性好，付费功能提示不突兀
4. 与 Notion 等成熟产品对齐，学习成本低

### 3.3 间距系统

#### 基础单位：4px

```
4px → 最小间距（图标与文字）
8px → 紧凑间距（相关元素）
12px → 标准间距（组件内）
16px → 舒适间距（组件间）
24px → 大间距（分区）
32px → 超大间距（页面级）
48px → 呼吸间距（英雄区域）
64px → 页面边距（桌面）
```

#### 常用间距组合

| 场景 | 间距值 | 示例 |
|------|--------|------|
| 图标与文字 | 8px | 🔍 搜索框 |
| 表单字段间 | 16px | 输入框间距 |
| 卡片内边距 | 16px | 内容到边框 |
| 卡片之间 | 12px | 列表项间距 |
| 分区之间 | 32px | 内容区块 |
| 页面边距 | 16px (移动) / 24px (桌面) | 屏幕边缘 |

### 3.6 动画规范

#### 时长

| 动画类型 | 时长 | 缓动 |
|---------|------|------|
| 微交互（hover） | 150ms | ease |
| 中等动画（展开） | 200ms | ease-out |
| 大动画（页面切换） | 300ms | ease-out |
| 复杂动画（录音脉冲） | 1500ms | ease-in-out |

#### 录音按钮脉冲动画

```css
@keyframes pulse-ring {
  0% { transform: scale(1); opacity: 1; }
  100% { transform: scale(1.6); opacity: 0; }
}

@keyframes pulse-core {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.05); }
}

/* 录音中应用 */
.recording {
  animation: pulse-core 1.5s ease-in-out infinite;
}

.recording::before {
  content: '';
  position: absolute;
  animation: pulse-ring 1.5s ease-out infinite;
}
```

### 5.4 付费弹窗设计规范

#### 尺寸
- 移动端：满屏（带关闭按钮）
- 平板/桌面：最大宽度 480px，居中

#### 动画
- 进入：fade-in + slide-up (300ms)
- 退出：fade-out (200ms)

#### 关闭方式
- 点击背景关闭
- 点击右上角 X
- 点击"暂不需要"

#### 频率控制
- 同一用户 7 天内最多显示 3 次
- 完成付费后不再显示
