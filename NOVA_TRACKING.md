### Context Graph 数据模型

```typescript
// 决策痕迹
interface ContextTrace {
  id: string;
  noteId: string;
  trigger: 'manual' | 'scheduled' | 'ai_suggested';
  inputs: string[];
  prompt?: string;
  generatedAt: string;
  model: string;
}

// 用户先例偏好
interface UserPrecedent {
  id: string;
  category: 'format' | 'tone' | 'structure' | 'voice';
  pattern: string;
  frequency: number;
  lastUsed: string;
}

// 跨系统关联
interface CrossSourceLink {
  id: string;
  sourceIds: string[];
  type: 'time_window' | 'semantic' | 'manual';
  confidence: number;
  createdAt: string;
}
```

## 🐛 Bug 记录

### BUG-001: 网站链接抓取后跳转逻辑错误
- **发现时间**: 2026-03-10
- **严重程度**: 🔴 P1 - 高
- **描述**: 从多模态捕获菜单输入网站链接并提交后，没有跳转到新建的笔记详情页，而是打开了 AI 助手浮层
- **复现步骤**:
  1. 点击右下角 ➕ 按钮打开多模态捕获菜单
  2. 选择"网站"选项
  3. 输入微信公众号链接（如 https://mp.weixin.qq.com/s/xxx）
  4. 点击提交按钮
  5. 观察到 AI 助手浮层打开，而非跳转笔记详情
- **期望行为**: 提交后应自动跳转到新创建的笔记详情页
- **实际行为**: 打开了 AI 助手浮层
- **负责人**: @Bernard
- **状态**: ⏳ 待修复

### BUG-002: 微信公众号内容抓取失败
- **发现时间**: 2026-03-10
- **严重程度**: 🟡 P2 - 中
- **描述**: 微信公众号文章有反爬机制，直接抓取返回 Parameter error 和 UI 提示文案，无法获取正文内容
- **错误内容示例**: `Parameter error : ，，，，，，，，，，，，.Video Mini Program Like，轻点两下取消赞 Wow，轻点两下取消在看`
- **调研结果**:
  - Cubox 使用「企业微信会话存档」API 实现（https://developer.work.weixin.qq.com/document/path/91774）
  - 该 API 需付费，2022年12月起 Cubox 将其升级为 Pro 会员专享功能
  - 其他方案：Jina AI / Firecrawl / 自建 Headless 浏览器
- **建议方案**: 
  - 短期: 接入 Jina AI 解析服务（快速实现）
  - 中期: 评估 Firecrawl 等第三方服务
  - 长期: 评估企业微信方案成本
- **负责人**: @Bernard
- **状态**: ⏳ 待评估方案
