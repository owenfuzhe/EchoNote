# EchoNote 重构追踪 v2.0

**项目**: EchoNote AI-Native 笔记应用重构  
**日期**: 2026-03-08  
**状态**: 🟡 UI 重构中  

---

## 原始需求（来自 Owen）

1. 方便记录想法、感兴趣的信息
2. 快速消化、理解爆炸的信息
3. 持续探索学习，形成体系性知识结构
4. 结构化规划整理，方便筛选查看管理
5. 识别信息关联，发现兴趣/想法/行为模式
6. Context 能准确传达给其他 AI Agent，成为 context 汇总/维护中枢

---

## v2.0 架构变化

| v1.0 | v2.0 |
|------|------|
| 3 Tabs (Home/Library/Explore) | 2 Screens (Home/Library) |
| Explore 页面 | 移除，技能改为长按触发 |
| 手动文件夹 | 语义自动聚类 |
| 静态加载 | 呼吸文字效果 |
| 卡片边框 | 纯白+留白层次 |
