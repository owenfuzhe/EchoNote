# EchoNote UI/UX 设计规范 v2.0

**版本：** v2.0  
**日期：** 2026-03-01  
**状态：** 定稿（按 Craft + Notion 风格重构）  
**图标库：** [Lucide Icons](https://lucide.dev/)  

---

## 1. 设计原则

### 1.1 核心理念
- **多模态输入**：语音、文字、链接、文件，统一汇聚
- **极简主义**：纯白背景、统一图标、无多余元素
- **AI 驱动**：智能标签、摘要、待办提取
- **行动导向**：不只是记录，更要推动执行

### 1.2 设计关键词
| 关键词 | 含义 | 体现 |
|--------|------|------|
| **Clean** | 干净利落 | Craft 式纯白、大量留白 |
| **Unified** | 高度统一 | Lucide 图标、单一配色 |
| **Smart** | 智能感知 | AI 自动分类、摘要 |
| **Actionable** | 行动导向 | 待办提取、行动 Tab |

## 2. 设计系统

### 2.1 颜色系统

- **图标统一**：所有图标使用 `gray-600`（激活时 `indigo-600`）
- **语音突出**：仅录音按钮可使用 `red-500` 作为核心功能强调
- **纯白背景**：参考 Craft，主背景纯白，减少视觉负担

## 7. 交互流程

### 7.1 录音流程

```
用户旅程:

1. 点击底部 [🎙️] 语音按钮（左三，红色）
   → 底部弹出录音模态框
   
2. 首次点击中央麦克风
   → 麦克风变绿色
   → 周围出现绿色脉冲光晕
   → 开始录音，显示时长
   → 可选：显示实时转录
   
3. 再次点击麦克风（录音中）
   → 暂停录音
   → 麦克风变黄色/橙色
   → 光晕消失
   → 显示"继续录音"
   
4. 点击「完成」
   → 停止录音
   → 进入笔记编辑页（AI已生成摘要、标签）
   
5. 点击「取消」
   → 放弃当前录音
   → 关闭模态框，返回原页面

状态流转:
  未录音(灰) → 点击 → 录音中(绿+光晕) → 点击 → 暂停(黄)
     ↑                                                ↓
     └──────────── 再次点击继续录音 ────────────────────┘
```
