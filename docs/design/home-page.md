# EchoNote 主页设计

> 设计师：Designer Agent  
> 日期：2026-03-03  
> 版本：v2.1（修正底部输入栏）

---

## 📱 设计概述

主页是 EchoNote 的核心页面，用户在这里查看今日动态、快速概览笔记。采用 Craft 风格设计，以卡片和内容为中心。

### 设计原则
- **内容优先**：减少视觉干扰，突出笔记内容
- **快速输入**：底部 5 按钮快速选择输入方式
- **浏览友好**：2 列网格展示更多最近笔记
- **极简美学**：参考 Craft 的干净布局

## 🎭 关键交互说明

### 1. 快速开始卡片

| 触发 | 用户点击"快速开始"卡片 |
|------|----------------------|
| 反馈 | 卡片展开显示 3 个选项 |
| 选项 | 开始录音 / 文字笔记 / 导入文件 |
| 动画 | 高度展开 200ms ease-out |

### 2. 底部导航 ➕ 录音按钮

| 触发 | 点击底部导航 ➕ 图标 |
|------|---------------------|
| 反馈 | 跳转到录音页（独立页面） |
| 状态 | 录音页高亮显示 |
| 逻辑 | 与首页分离，专注录音体验 |

### 3. 最近笔记交互

| 操作 | 触发 | 结果 |
|------|------|------|
| **查看** | 点击笔记卡片 | 跳转到笔记详情页 |
| **播放** | 点击播放图标 | 内嵌播放该录音 |
| **更多** | 长按卡片 | 显示操作菜单（重命名、删除、分享） |
| **刷新** | 下拉列表 | 刷新最近笔记列表 |

### 4. 搜索框

| 操作 | 触发 | 结果 |
|------|------|------|
| **搜索** | 输入文字 | 实时过滤笔记列表 |
| **清空** | 点击 ✕ 图标 | 清空搜索内容 |
| **聚焦** | 点击搜索框 | 键盘弹出，显示搜索历史 |

### 4. 底部输入栏 (BottomInputBar) - Craft 风格分离式悬浮

> ⚠️ **重要设计变更：**
> - ❌ **不是** 固定在底部的导航栏
> - ❌ **不是** 连在一起的胶囊
> - ✅ **是** 左三右二分离式悬浮（Craft 风格）
> - 页面切换 → 通过顶部 Tab 完成
> - 底部按钮 → 选择输入方式（文字/语音/文件等）

```
┌─────────────────────────────────┐
│  内容区域                       │
│  ┌─────────────────────────┐    │
│  │ 笔记卡片                 │    │
│  └─────────────────────────┘    │
│  ┌─────────────────────────┐    │
│  │ 笔记卡片                 │    │
│  └─────────────────────────┘    │
│                                 │
│                                 │
│  ┌───────────────┐   ┌───────┐  │  ← 左三    右二
│  │ Plus SquarePen│   │Search │  │     分离式
│  │    Mic        │   │  Bot  │  │     悬浮
│  └───────────────┘   └───────┘  │
│                                 │
│  ─────────────────────────────  │  ← 底部安全区
└─────────────────────────────────┘
```

**分离式悬浮规格：**

```
左侧组（3 按钮）              右侧组（2 按钮）
┌─────────────────┐         ┌─────────┐
│ Plus  SquarePen │         │ Search  │
│     Mic         │         │   Bot   │
└─────────────────┘         └─────────┘
• 位置：左下角               • 位置：右下角
• 宽度：约 40% 屏幕宽          • 宽度：约 25% 屏幕宽
• 间距：距离底部 16dp         • 间距：距离底部 16dp
• 左右边距：16dp             • 右边距：16dp
• 背景：白色                  • 背景：白色
• 圆角：9999px (胶囊状)        • 圆角：9999px (胶囊状)
• 阴影：0 4px 12px rgba(0,0,0,0.15) • 阴影：0 4px 12px rgba(0,0,0,0.15)
```

**按钮配置：**

> ⚠️ **图标必须严格按 ui-spec.md 第 3.2 节使用，不可替换！**

**左侧组（3 按钮）：**

| 位置 | Lucide Icon | 标签 | 颜色 | 功能 |
|------|-------------|------|------|------|
| 左一 | `Plus` | 更多 | gray-600 | 更多选项 |
| 左二 | `SquarePen` | 文字 | gray-600 | 文字笔记 |
| 左三 | `Mic` | 语音 | red-500 | 录音（视觉中心） |

**右侧组（2 按钮）：**

| 位置 | Lucide Icon | 标签 | 颜色 | 功能 |
|------|-------------|------|------|------|
| 右一 | `Search` | 搜索 | gray-600 | 搜索 |
| 右二 | `Bot` | AI | indigo-600 | AI 助手 |

**图标来源：** [Lucide Icons](https://lucide.dev/icons/)  
**规范文档：** `ui-spec.md` 第 3.2 节 底部导航图标

**设计说明：**
- 左三右二分离式布局（Craft 风格）
- 两组之间有明显间距
- `Mic` 语音按钮在左侧组居中，是视觉中心
- `Mic` 使用红色突出核心功能
- `Bot` AI 使用靛蓝色区分
- 录音中时 `Mic` 有绿色脉冲光晕动画
- **所有按钮都不跳转页面，都是弹出模态框或直接操作**
