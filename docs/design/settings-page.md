## 2. 组件规范

### 2.1 分组标题
| 属性 | 值 |
|------|-----|
| 字号 | 14px |
| 字重 | 600 |
| 颜色 | `gray-600` |
| 间距 | 上 24px, 下 8px |
| 左边距 | 16px |

### 2.2 设置卡片
| 属性 | 值 |
|------|-----|
| 背景 | 纯白 |
| 边框 | 1px `gray-200` |
| 圆角 | 12px |
| 内边距 | 16px |
| 间距 | 12px |

### 2.3 设置项
| 元素 | 样式 |
|------|------|
| 标签 | 14px, `gray-800`, 400 |
| 描述 | 12px, `gray-500`, 400 |
| 开关 | 28px × 16px, `indigo-600` (开启) |
| 下拉 | 14px, `gray-600`, 右侧箭头 |

### 2.4 进度条
| 属性 | 值 |
|------|-----|
| 高度 | 4px |
| 背景 | `gray-200` |
| 前景 | `indigo-600` |
| 圆角 | 2px |
| 文字 | 12px, `gray-500` |

### 2.5 退出登录按钮
| 属性 | 值 |
|------|-----|

| 文字 | 14px, `gray-600` |
| 悬停 | 背景 `gray-50` |

## 3. 交互流程

### 3.1 账户管理
```
点击账户卡片 → 跳转账户详情页 → 管理订阅/头像/密码
```

### 3.2 存储管理
```
点击管理存储 → 查看存储详情 → 清理/删除/升级
```

### 3.3 AI 模型选择
```
点击下拉 → 选择模型 (GLM-4/Kimi/通义) → 保存设置
```

### 3.4 主题切换
```
点击下拉 → 选择主题 (浅色/深色/自动) → 即时应用
```
