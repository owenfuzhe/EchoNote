## 📱 设计概述

笔记列表页是 EchoNote 的核心内容页面，展示用户所有的语音笔记，支持搜索、筛选、播放和管理操作。

### 设计原则
- **高效浏览**：快速定位目标笔记
- **即时操作**：播放、编辑一键可达
- **批量管理**：支持多选和批量操作
- **友好空态**：无笔记时提供明确引导

## 🎭 关键交互说明

### 1. 搜索功能

| 触发 | 点击搜索栏或下拉手势 |
|------|---------------------|
| 状态 | 展开搜索模式，显示历史记录 |
| 输入 | 实时过滤列表，高亮匹配文字 |
| 清除 | 点击右侧 ✕ 清空搜索词 |

**搜索维度：**
- 标题匹配
- 标签匹配
- 时间范围（今天、本周、本月）

**交互细节：**
- 输入时 300ms 防抖后执行搜索
- 无结果时显示"未找到相关笔记"
- 支持搜索历史快捷选择

### 2. 标签筛选

| 触发 | 点击标签 Pill |
|------|--------------|
| 单选 | 点击切换当前筛选标签 |
| 全部 | 点击"全部"清除筛选 |
| 视觉 | 选中标签高亮（indigo-600 背景） |

**标签显示：**
- 默认显示前 3-4 个常用标签
- 更多标签可横向滑动查看

### 3. 播放语音

| 触发 | 点击播放按钮 ▶️ |
|------|----------------|
| 状态变化 | ▶️ → ⏸️，展开迷你播放器 |
| 进度 | 显示当前播放时间和进度条 |
| 波形 | 迷你波形随播放动态跳动 |
| 切换 | 播放新笔记时自动停止当前 |

**播放器状态：**
```
空闲:    [▶️] 点击播放
播放中:  [⏸️] 点击暂停 + 显示进度条
暂停:    [▶️] 点击继续 + 进度条保持
```

### 4. 长按/右键菜单

| 触发 | 长按列表项（移动端）/ 右键点击（桌面端） |
|------|----------------------------------------|
| 菜单 | 从底部滑出（移动端）/ 右键菜单位置（桌面） |
| 选项 | 编辑标题、管理标签、分享、删除 |
| 关闭 | 点击遮罩层或取消按钮 |

**菜单项详情：**

| 选项 | 图标 | 操作 |
|------|------|------|
| 编辑标题 | ✏️ | 弹出编辑对话框 |
| 管理标签 | 🏷️ | 跳转标签管理页 |
| 分享 | 📤 | 唤起系统分享面板 |
| 删除 | 🗑️ | 弹出二次确认 |

### 5. 下拉刷新

| 触发 | 列表顶部下拉 |
|------|-------------|
| 阈值 | 下拉距离 > 80px |
| 提示 | "↓ 释放刷新" → "🔄 加载中..." |
| 完成 | 刷新完成后自动收回 |
| 动画 | 旋转指示器 + 渐隐 |

### 6. 加载更多

| 触发 | 滚动到底部 |
|------|-----------|
| 提示 | 显示"↓ 加载更多" |
| 加载 | 自动加载下一页（20条/页） |
| 完成 | 无更多数据时显示"没有更多了" |
| 错误 | 加载失败显示"点击重试" |

### 7. 空状态交互

| 触发 | 列表为空时自动显示 |
|------|-------------------|
| 图标 | 📝🎙️ 组合动画 |
| 引导 | "创建你的第一条语音笔记" |
| 按钮 | "去录音" → 跳转首页 |
| 动效 | 图标轻微浮动，吸引注意 |

| Token | 值 | 用途 |

| 断点 | 宽度 | 列表列数 |
