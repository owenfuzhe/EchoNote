# EchoNote 修复任务清单

## 任务 1: 搜索页面优化 + 智能搜索引擎

### 1.1 搜索页面尺寸调整
- 当前：全屏 `fixed inset-0`
- 目标：和 APP 一样的移动端尺寸 `max-w-md`
- 位置：在底部导航上方显示，不要覆盖整个屏幕

### 1.2 内置智能搜索引擎
- 使用 Fuse.js 实现客户端全文搜索
- 支持模糊匹配、权重排序、结果高亮
- 参考文档：`docs/search-engine-design.md`

**文件修改：**
- `src/components/SearchView.tsx` - 重构搜索页面
- `src/services/search-engine.ts` - 新建搜索引擎服务
- `src/App.tsx` - 调整搜索页面显示方式
